 SAS Clinical Trial Project

## Objective
This project analyzes blood pressure reduction in patients using SAS.

## Dataset
Clinical trial dataset containing:
- Patient demographics
- Treatment groups
- Blood pressure measurements

## SAS Procedures Used
- PROC IMPORT
- PROC PRINT
- PROC MEANS
- PROC FREQ
- PROC SGPLOT

## Skills Demonstrated
- Data importing
- Data cleaning
- Statistical analysis
- Frequency analysis
- Data visualization

## Project Output
The analysis showed blood pressure reduction differences between treatment groups.

## Tools Used
- SAS Studio
- GitHub
